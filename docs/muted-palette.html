Token Changes for Muted Aurora

To implement the muted palette, we'd update these tokens:

--cp-accent: #1a9e8f (was #2dd4bf)

--cp-pink: #d66ba0 (was #f472b6)

--cp-violet: #9678e0 (was #a78bfa)

The glow tokens would automatically use these new values since they reference the base colors.
